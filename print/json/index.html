জেসন (JSON) এর পূর্নরুপ হচ্ছে জাভাস্ক্রিপ্ট অবজেক্ট নোটেশন (JavaScript Object Notation)। এটা হালকা একটা ডাটা রিপ্রেজেন্ট করার ফরম্যাট। আমরা সহজেই এর সাহায্যে ডাটা পড়তে বা লিখতে পারি এবং একই সাথে ম্যাশিনের জন্যও লিখা বা পড়া সহজ। এটা জাভাস্ক্রিপ্ট বেসড একটা ফরম্যাট (জাভাস্ক্রিপ্ট স্ট্যান্ডার্ড ECMA-262 তৃতীয় এডিশন - ডিসেম্বর ১৯৯৯)
