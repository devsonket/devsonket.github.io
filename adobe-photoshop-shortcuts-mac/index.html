অ্যাডোবি ফটোশপ(ম্যাক)

অ্যাডোবি ফটোশপের এর প্রয়োজনীয় ম্যাক শর্টকাট কী

প্রিন্ট করুন

ফাইল (File)

Link

  • নতুন ফাইল (New)

    Cmd + N

  • ফাইল ওপেন করতে (Open)

    Cmd + O

  • ফাইল এডোবি ব্রিজে ব্রাউজ করতে (Browse in Bridge)

    Opt + Cmd + O or Shift + Cmd + O

  • ফাইল বন্ধ করা (Close)

    Cmd + W

  • সকল ফাইল বন্ধ করতে (Close All)

    Opt + Cmd + W

  • ফাইল বন্ধ করুন এবং এডোবি ব্রিজে যান (Close and Go to Bridge)

    Shift + Cmd + W

  • ফাইল সেভ করতে (Save)

    Cmd + S

  • ফাইল ফরম্যাট হিসেবে সেভ করতে (Save As)

    Shift + Cmd + S or Opt + Cmd + S

  • ওয়েব এর জন্য ফাইল সেভ করতে (Save for Web)

    Opt + Shift + Cmd + S

  • ফাইল আগেকার অবস্থায় আনতে (Revert)

    F12

  • ফাইল সম্পর্কিত তথ্য (File Info)

    Opt + Shift + Cmd + I

  • ফাইল প্রিন্ট করতে (Print)

    Cmd + P

  • এক কপি ফাইল প্রিন্ট করতে (Print One Copy)

    Opt + Shift + Cmd + P

    Shift + Cmd + C

  • পেস্ট (Paste)

  • পূরণ করা (Fill)

    Shift + F5

  • Content-Aware স্কেল সিলেক্ট করতে (Content-Aware Scale)

    Opt + Shift + Cmd + C

  • ফ্রি ট্রান্সফর্ম (Free Transform)

    Cmd + T

  • কালার সেটিংস (Color Settings)

    Shift + Cmd + K

  • কীবোর্ড শর্টকাটগুলি (Keyboard Shortcuts)

    Opt + Shift + Cmd + K

  • মেনু (Menus)

    Opt + Shift + Cmd + M

  • প্রেফারেন্সেস > জেনারেল (Preferences > General)

    Cmd + K

ইমেজ (Image)

Link

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে লেভেল সিলেক্ট কি (Adjustments > Levels)

    Cmd + L

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে কার্ভস সিলেক্ট কি (Adjustments > Curves)

    Cmd + M

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে হিউ/স্যাচুরেশন সিলেক্ট কি (Adjustments > Hue/Saturation)

    Cmd + U

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে কালার ব্যালান্স সিলেক্ট কি (Adjustments > Color Balance)

    Cmd + B

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে সাদা কালো সিলেক্ট কি (Adjustments > Black & White)

    Opt + Shift + Cmd + B

  • ইমেজ অ্যাডজাস্টমেন্ট থেকে ইনভার্ট সিলেক্ট কি (Adjustments > Invert)

    Cmd + I

  • ইমেজ এর কালার কে গ্রেস্কালে কনভার্ট করতে (Desaturate)

    Shift + Cmd + U

  • ইমেজ এর অটো টোন কি (Auto Tone)

    Shift + Cmd + L

  • ইমেজ এর অটো কনট্রাস্ট কি (Auto Contrast)

    Opt + Shift + Cmd + L

  • ইমেজ এর অটো কালার কি (Auto Color)

    Shift + Cmd + B

  • ইমেজ সাইজ কি (Image Size)

    Opt + Cmd + I

  • ইমেজ ক্যানভাস কি (Canvas Size)

    Opt + Cmd + C

লেয়ার (Layer)

Link

  • নতুন লেয়ার (New > Layer)

    Shift + Cmd + N

  • নতুন লেয়ার কপি করতে (New > Layer via Copy)

    Cmd + J

  • নতুন লেয়ার কাট করতে (New > Layer via Cut)

    Shift + Cmd + J

  • ক্লিপিং মাস্ক বানানো/রিলিজ করতে (Create/Release Clipping Mask)

    Opt + Cmd + G

  • লেয়ার গ্রুপ করতে (Group Layers)

    Cmd + G

  • লেয়ার আন গ্রুপ করতে (Ungroup Layers)

    Shift + Cmd + G

  • লেয়ার ফ্রন্ট এ আনতে (Arrange > Bring to Front)

    Shift + Cmd + ]

  • লেয়ার সামনে এ আনতে, এক্ষেত্রে লেয়ার এক ধাপ করে সামনে আসে (Arrange > Bring Forward)

    Cmd + ]

  • লেয়ার কে পিছনে নিতে, এক্ষেত্রে লেয়ার এক ধাপ করে পিছনে যায় (Arrange > Send Backward)

    Cmd + [

  • লেয়ার কে পিছনে নিতে (Arrange > Send to Back)

    Shift + Cmd + [

  • সিলেক্ট করা লেয়ার মার্জ করতে (Merge Layers)

    Cmd + E

  • ভিসিবল লেয়ার মার্জ করতে (Merge Visible)

    Shift + Cmd + E
